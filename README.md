# API de Registro e Inicio de Sesión en PHP

## Descripción
Este servicio web permite registrar e iniciar sesión con usuarios usando PHP y MySQL.

## Endpoints
- POST `/register.php`  
  Cuerpo JSON: `{ "usuario": "nombre", "contrasena": "1234" }`

- POST `/login.php`  
  Cuerpo JSON: `{ "usuario": "nombre", "contrasena": "1234" }`
